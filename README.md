# 简介
多条件筛选菜单，在020app 上类似选地区，选择类型等功能。
一般早先来说，都是用popupwindow 去实现这个功能。
但其实原生的布局去实现这样一个功能同样是可以的，并且可以扩展定制样式。

基本所有的菜单，都可以归纳为以上两类：1 Listview(单列) 2 Gridview(多列)

ps:可以用Recycleview 去替换Listview Gridview。
